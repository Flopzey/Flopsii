# Flopsii
~~Your new best friend!~~ A bot for Discord.

## Control Center

The optional Swing interface shows the bot connection, account, guild count,
uptime and current presence. It can change the online status and custom
activity, display the latest application logs and shut the bot down safely.

Enable or disable it in `config.ini`:

```ini
[Setup]
UseGui = true
```

Closing the Control Center only closes the window. Use the
**Bot herunterfahren** button to disconnect the bot.
